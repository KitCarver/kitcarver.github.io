  1. Выбираем тип расчета с помощью горизонтального ползунка: “по длине” или “по массе”. Расчет “по длине” означает вычисление веса изделия, если известна его длина. Расчет “по массе” направлен на вычисление длины изделия, если известна его масса. По умолчанию установлено положение регулятора “по длине”.
  2. Выбираем марку стали в соответствующем выпадающем списке. У каждой стали своя плотность, что влияет на расчет требуемых параметров. По умолчанию выбрана сталь марки AISI 304.
  3. С помощью блока кнопок в левой части калькулятора выбираем вид металлопроката: круглую трубу, профильную трубу, лист, круг, квадрат, шестигранник, уголок или полосу. Кнопка выбранного изделия будет подсвечена светло-красным цветом. В центральном экране калькулятора появится изображение выбранного профиля с полями для ввода соответствующих параметров (диаметр, высота, ширина, толщина и т.д.). По умолчанию для расчетов выбрана круглая труба.
  4. Указываем требуемые параметры в соответствующих полях ввода. Чтобы избежать ошибки в расчетах, следует обратить внимание на единицы измерения, которые указаны для различных параметров. В полях ввода можно указывать только цифры, а также одну точку или запятую. Попытки ввести другие символы ни к чему не приведут. Для выполнения расчетов должны быть указаны все параметры, т.е. должны быть заполнены все поля ввода.
  5. Для начала расчета необходимо кликнуть левой кнопкой мыши по кнопке “РАССЧИТАТЬ”. В результате чего настоящая инструкция будет скрыта, а на ее месте появятся строка с исходными данными для расчета (вид металлопроката, марка стали, габаритные и/или весовые параметры) и результатом расчета (длина или вес изделия). При каждом новом расчете происходит сохранение всех предыдущих результатов. Строка с результатом последнего расчета располагается в самом верху перечня строк с результатами и выделана жирным шрифтом и темно-красным цветом, а строка первоначального расчета находится в самом низу данного перечня. Если перечень результатов расчетов содержит более трех строк, то с правой стороны этого перечня появляется полоса прокрутки, которая позволяет просмотреть все результаты расчетов.
  6. Для быстрой очистки полей ввода параметров, а также установки других условий расчета в начальное положение (по умолчанию) и удалению результатов предыдущих расчетов, предназначена кнопка “СБРОСИТЬ”.
  7. Для отправки результатов расчетов на почту пользователя предусмотрено поле ввода адреса электронной почты и соответствующая кнопка отправки. При этом пользователь может подписаться на получение новостей компании Инокспоинт.
  8. Если во время расчетов требуется открыть данную инструкцию, то следует нажать на кнопке со знаком вопроса, которая находится слева от перечня строк с результатами расчетов. В результате строки с результатами исчезнут, а на их месте появится искомая инструкция. При этом кнопка со знаком вопроса будет заменена на кнопку с крестиком, которая необходима для сворачивания инструкции и возврата к перечню строк с результатами расчетов.
  9. Все замечания по работе калькулятора следует отправлять на электронную почту info@inoxpoint.kz
